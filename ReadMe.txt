1.需要先安装python3.0以上版本，配置python环境
2.配置pip安装工具
3.安装mysql 客户端和服务端
   (创建数据库，在客户上创建，setting-数据库名）
   切换到根目录执行创建表
   python manage.py makemigrations
   python manage.py migrate
4、加载三方库，pip install -r requirements.txt
5.下载pycharm 开发工具
6.配置pycharm debug 环境 python manage.py runserver
7.点击run
8.打开网页：127.0.0.1:8000
